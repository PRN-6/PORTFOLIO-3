# Modern Portfolio

A modern, interactive portfolio website built with React, featuring responsive design, and advanced interactive features. This project showcases professional web development skills with cutting-edge technologies and smooth user experiences.

## 🚀 Features

### Core Features
- **Responsive Design**: Fully responsive layout that works seamlessly on all devices
- **Interactive Navigation**: Smooth hamburger menu with mobile support
- **GSAP Animations**: Smooth, professional animations throughout the site
- **Click Counter**: Real-time click tracking 

### Technical Highlights
- **Modern Stack**: React 19, Vite, Tailwind CSS
- **Performance Optimized**: Fast loading with optimized assets
- **Component Architecture**: Clean, modular React components

## 🛠️ Tech Stack

### Frontend
- **React 19** - Modern React with hooks and concurrent features
- **Vite** - Lightning-fast build tool and dev server
- **Tailwind CSS** - Utility-first CSS framework for rapid styling
- **GSAP** - Professional animation library
- **React Router** - Client-side routing

### UI/UX
- **Lucide React** - Beautiful icon library
- **React Icons** - Comprehensive icon collection
- **DaisyUI** - Component library built on Tailwind CSS

### Development Tools
- **ESLint** - Code linting and formatting
- **PostCSS** - CSS processing

## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn package manager

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/PRN-6/3D-PORTFOLIO.git
   cd 3D-PORTFOLIO
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:5173`

### Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
